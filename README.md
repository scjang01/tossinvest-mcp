# tossinvest-mcp

토스증권 공식 오픈 API 지원 Model Context Protocol (MCP) 서버입니다.

현재 토스증권 오픈 API 출시에 맞추어 공식 규격 대응을 준비 중인 프로젝트입니다. 정식 API가 릴리즈되고 키 발급이 시작되면 연동 기능이 업데이트될 예정입니다.

## 주요 제공 예정 기능

Claude, GPT, Gemini 등 MCP를 지원하는 AI를 통해 다음과 같은 작업을 수행할 수 있도록 도구(Tool)를 제공할 예정입니다.

- 계좌 정보 및 잔고 조회
- 실시간 국내/해외 주식 시세 및 차트 조회
- 주식 매수 및 매도 주문 실행
- 미체결 주문 확인 및 취소

## 설치 및 사용법 (준비 중)

정식 릴리즈 후 다음과 같은 명령어로 간단히 실행하거나 설정 파일에 등록하여 사용할 수 있도록 패키징될 예정입니다.

```bash
npx tossinvest-mcp
```
